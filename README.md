# Gosh

A simple Unix-like shell written in Go.

This project is built mainly for learning purposes while following the **CodeCrafters “Build Your Own Shell” challenge** as a reference. The goal is to understand how shells work internally—command parsing, built-ins, and process execution.

It focuses on understanding **system calls**, the **OS process model**, and Go’s **`os` and `os/exec` standard packages** by implementing shell behavior from scratch.
